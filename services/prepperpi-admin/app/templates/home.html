  • Network

    Change the Wi-Fi name (SSID), set or remove a password, pick a channel, and set your regulatory country code.

  • Storage and health

    Live CPU, RAM, temperature, and disk-usage stats. Per-USB write toggle, recent event log, and a downloadable diagnostics tarball.

  • Content catalog

    Browse the Kiwix library and download ZIMs in the background. Pause / resume / cancel from the same page; downloads can target the SD card or any writable USB drive.

  • Offline maps

    List installed map regions, see total size and zoom range, and uninstall ones you don't need. The live map is at /maps/.

Uplink: plug Ethernet to bring the Pi online for downloads. AP clients are not bridged to the internet (the Pi is not a hotspot).

The admin console is reachable only from clients on the PrepperPi Wi-Fi network or from the device itself. It is never exposed to an upstream uplink.
