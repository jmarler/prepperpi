Admin overview

Manage the parts of your PrepperPi that don't belong on the public landing page.

  • Network

    Change the Wi-Fi name (SSID), set or remove a password, pick a channel, and set your regulatory country code.

  • Storage (soon)

    Per-USB-mount write toggle, disk usage, content folders. Lands with E4-S2.

  • Content (soon)

    Browse the Kiwix catalog, queue downloads, manage installed ZIMs. Lands with E2-S3.

  • Online mode (soon)

    Connect a temporary uplink for content downloads or system updates, then return to fully offline. Lands with E4-S3.

The admin console is reachable only from clients on the PrepperPi Wi-Fi network or from the device itself. It is never exposed to an upstream uplink.
